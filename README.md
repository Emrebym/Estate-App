# 🏠 Estate App

A real estate listing application for browsing, searching, and viewing property details including images, pricing, and location info.

## Features

- **Property listings** — browse available houses and apartments
- **Detail pages** — view property images, specs, and pricing
- **Search & filter** — find properties by location, price range, or type
- **Map integration** — see property locations on a map
- **Responsive design** — mobile-friendly interface
